# Brand System

The platform UI uses a restrained teal SaaS palette. Public lead magnet pages inherit each account's uploaded logo and color choices while preserving the original Diandra-style page composition.

## Account Colors

- Primary color: headings, CTAs, accents
- Accent color: subtle panel gradients
- Success color: confirmation accents

## Public Page Style

Public pages keep:

- centered logo header
- large gradient headline
- two-column landing page layout
- image preview
- rounded lead capture form
- branded confirmation state
